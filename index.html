What is LCode?

LCode is an open-source standard for assigning short, easy-to-share alphanumeric codes to every plot, building, or location in the world. Instead of confusing long GPS coordinates or vague addresses, a single code points precisely to a spot â€” forever.

How LCode Works

LCode uses a universal format:

CCC-SSS-XXX-YYYYY

CCC: Country code
SSS: State or province
XXX: Local area code
YYYYY: Unique identifier for the property or plot
