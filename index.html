📜 What is LCode?

LCode is an open-source, global code system that assigns short, easy-to-share alphanumeric codes to every plot, building, or location in the world. Instead of long GPS coordinates or confusing addresses, one simple code identifies any location — forever.

🏗 How It Works

The standard LCode format is:

CCC-SSS-XXX-YYYYY
  • CCC → Country Code
  • SSS → State/Province Code
  • XXX → Local Area Code
  • YYYYY → Unique property identifier

Example: 001-018-323-X5A2J → USA → California → Los Angeles → Exact plot.
